## c01-data-transformation

This component contains code and notebooks for transforming raw or intermediate data into analysis-ready datasets.

- **Typical contents**: data cleaning scripts, feature engineering, integration of multiple sources, and reproducible transformation pipelines.
- **Inputs**: files from the `data` folder (e.g., `raw/` or `interim/`).
- **Outputs**: cleaned and structured data written back to `data` (e.g., `processed/`), to be used by later components.
